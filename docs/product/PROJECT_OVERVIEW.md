# AUREX

## Project Overview

AUREX (AI-powered Banking Change Analysis Agent) is an intelligent software engineering assistant designed for banking application development teams.

Its purpose is to help developers, business analysts, solution architects, and QA engineers analyze banking change requests before implementation.

Instead of manually identifying impacted modules, workflows, risks, and development tasks, AUREX performs an initial AI-assisted analysis and presents the information in a structured dashboard.

The goal of AUREX is to improve requirement analysis, reduce manual effort, and help software teams begin implementation with better clarity.

---

## Project Vision

Build a modern AI-powered engineering platform that assists banking software teams during requirement analysis and project planning.
