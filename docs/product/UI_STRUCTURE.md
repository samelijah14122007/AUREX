# AUREX UI Structure

## Page 1 - Landing Page

Purpose:
Introduce AUREX and its capabilities.

Components:
- Navigation Bar
- Hero Section
- Product Overview
- Features
- Technology Stack
- Footer
- "Get Started" Button

-----------------------------------------

## Page 2 - Dashboard

Purpose:
Central workspace for the user.

Components:
- Sidebar
- Welcome Card
- New Analysis Button
- Recent Analyses
- Statistics Cards

-----------------------------------------

## Page 3 - New Analysis

Purpose:
Submit a banking change request.

Components:
- Large Text Area
- Example Prompt
- Analyze Button
- Clear Button

-----------------------------------------

## Page 4 - Analysis Result

Purpose:
Display AI-generated analysis.

Sections:

1. Requirement Summary

2. Banking Domain

3. Impacted Modules

4. Workflow

5. Risk Analysis

6. Development Checklist

7. Testing Checklist

8. Complexity Score

9. Download Report

-----------------------------------------

## Page 5 - History

Purpose:
Display previously generated analyses.

Components:
- Search
- Filter
- Analysis Cards
- View Report

-----------------------------------------
